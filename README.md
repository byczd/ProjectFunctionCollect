# README

ProjectFunctionCollect 将一些平时的常用功能整合起来现在包含有

* 首页 —— 将数字转换成 33,333,333.00 的样式 
* 图片 —— 图片预览 
* 弹窗 —— 系统弹窗、自定义弹窗、基于 MBProgressHUD 封装的弹窗 
* 我的 —— 组织架构多选、组织架构单选
